# Description

Nextjs para consumir servicio de Alpha Vantage

- Nextjs
- Hooks
- Redux
- Rxjs
- Material-UI
- AMChart

## How to use

Repo

- [https://github.com/isra/alpha-vantage-api-nextjs.git](https://github.com/isra/alpha-vantage-api-nextjs.git)

Install it and run:

```sh
npm install
npm run dev
```
